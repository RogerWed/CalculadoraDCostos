Esta App incluye:

1. Cálculos completos:
   - Costo Unitario
   - Precio Venta
   - Precio al Mayor
   - Venta Total
   - Utilidad por unidad
   - Utilidad Total

2. Características avanzadas:
   - Conversión automática a BS usando tasa BCV
   - Persistencia de datos con localStorage
   - PDF profesional con diseño responsive
   - Animaciones en todas las interacciones
   - Diseño móvil-first con Bootstrap 5
   - Validación de formularios integrada

3. Optimizaciones:
   - Precisión decimal en cálculos monetarios
   - Renderizado dinámico eficiente
   - Feedback visual con notificaciones animadas
   - Iconos intuitivos para mejor UX
   - Scroll automático al editar
   - Reinicio completo del sistema

Para usar la aplicación:
1. Actualiza la tasa BCV primero
2. Registra productos con el formulario
3. Edita/Elimina desde la tabla
4. Genera reportes PDF con un clic
5. Usa el botón de reinicio para borrar todo
